# SteelSeries GG Gaming Presets

Community-contributed presets, EQ configs and profiles optimized for gaming audio (Sonar) and device customization (Engine) in **SteelSeries GG**.

> These are user-made presets only.  
> Always use the **official** SteelSeries GG from https://steelseries.com/gg

![SteelSeries GG Sonar Interface](https://images.ctfassets.net/w5r1fvmogo3f/O5S0VmvvVVSsyB9YPpTAI/868c4c3ee1c98096dba76e3c0ecb1485/GG_desktop_sonar_thumbnail.png)

## What's inside this repository

- 🎮 Sonar EQ & mixer presets (.json / .cfg files for Game/Chat/AUX channels)
- 🎧 Audio filter & spatial sound tweak templates
- 🔊 Low-latency & footsteps enhancement profiles
- 🖼️ Engine lighting / macro / DPI presets (for compatible devices)
- 📸 Example screenshots and usage notes

All files are plain text / configs — **no executables, no installers, no modifications** to SteelSeries GG itself.

## How to use

1. Download the latest release from the **Releases** tab on the right → choose the .zip you need  
2. Extract the archive to any convenient folder  
3. Open official **SteelSeries GG** (download from https://steelseries.com/gg if needed)  
4. For **Sonar**: Go to Sonar tab → Presets → Import / Apply manually (copy EQ values or load .json if supported)  
5. For **Engine**: Go to Engine tab → Device settings → Import profile / apply lighting/macro values  
6. Test in-game and adjust (e.g., per-game profiles in Sonar)  

**Tip**: Use built-in "Favorites" in Sonar for quick access to your best presets. Backup current settings before applying new ones!

## Important notes

- Requires **official SteelSeries GG** from https://steelseries.com/gg (free download)
- These presets are provided **as-is** — test them in a non-competitive game first
- Backup your existing Sonar/Engine configs before importing/applying
- Compatible with recent GG versions (Sonar 1.0+, Engine integrated) on Windows
